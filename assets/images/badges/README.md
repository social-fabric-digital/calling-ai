## Usage

The app will automatically select the appropriate badge based on:
1. User's goals and their categories (career, social, personal)
2. Progress metrics (streaks, consistency, completed goals)
3. User achievements and milestones

If a badge image is missing, the app will fallback to the default trophy image.
